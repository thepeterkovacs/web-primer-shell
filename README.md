> **Warning**

## Quick start

##### initialize with Next.js CLI tool

```bash
pnpm create next-app -e https://github.com/thepeterkovacs/web-primer-shell
```

##### navigate to app directory

```bash
cd my-app
```

##### rename .env.template to .env.local

```bash
mv .env.template .env.local
```
